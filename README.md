# Z80-AY-music-bell
Музыкальный звонок на звуковом сопроцессоре AY8910. Печатная плата отрисована мной по мотивам приведенной схемы в журнале Радио №6 за 1998 год и полностью соответствует схеме журнала, изменен номинал конденсатора С1 нужно использовать 10мкф, а также резистор R4 должен быть номиналом 4,7кОм.
 

Ссылка на проект oshwlab: https://oshwlab.com/svitalys/z80-ay
